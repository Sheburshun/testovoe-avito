# testovoe-avito

Репозиторий Амосовой Анжелы с тестовыми заданиями 1 и 2.2 для стажировки от Avito по направление QA Engineer. 


## Структура проекта

```text
TESTOVOE-AVITO/
├── Task1/
│   ├── bugsImages/                     # скриншоты багов из первого задания
│   └── BUGS_FROM_SCREENSHOT.md         # баг-репорты по дефектам со скриншота
├── Task2/
│   ├── bugsImages2/                    # скриншоты багов, найденных при тестировании второго задания
│   ├── pages/
│   │   ├── basePage.ts                 # базовый класс с общими методами для page object
│   │   ├── listPage.ts                 # page object главной страницы со списком объявлений
│   │   └── statsPage.ts                # page object страницы статистики
│   ├── tests/
│   │   ├── desktop/
│   │   │   ├── list.test.ts            # автотесты для desktop-сценариев страницы объявлений
│   │   │   └── stats.test.ts           # автотесты для desktop-сценариев страницы статистики
│   │   └── mobile/
│   │       └── theme.mobile.spec.ts    # автотест мобильного сценария переключения темы
│   ├── utils/
│   │   └── ...                         # вспомогательные утилиты, тестовые данные и хелперы
│   ├── BUGS.md                         # баг-репорты по найденным дефектам во втором задании
│   ├── package.json                    # зависимости проекта и npm-скрипты
│   ├── playwright.config.ts            # конфигурация Playwright
│   ├── TESTCASES.md                    # тест-кейсы по пользовательским сценариям
│   └── tsconfig.json                   # конфигурация TypeScript
├── .gitignore                          # список файлов и папок, которые не попадают в git
└── README.md                           # инструкция по запуску и описание проекта
```


## Как запустить проект

### 1. Скллонируйте репозиторий

Откройте терминал и выполните команду:

```bash
git clone https://github.com/Sheburshun/testovoe-avito.git
```

Либо скачайте проект в виде `.zip`-архива [по ссылке](https://github.com/Sheburshun/testovoe-avito.git) и распакуйте его на компьютере.

### 2. Перейдите в папку проекта

```bash
cd testovoe-avito
```

Затем перейдите в папку со второй частью задания:

```bash
cd Task2
```

### 3. Убедитесь, что у вас установлен Node.js

Проверьте версию Node.js:

```bash
node -v
```

Также желательно проверить версию `npm`:

```bash
npm -v
```

Если Node.js не установлен, скачайте и установите его с официального сайта.  
Во время установки можно оставить стандартные настройки.

### 4. Установите зависимости

В папке `Task2` выполните команду:

```bash
npm install
```

### 5. Установите браузеры Playwright

После установки зависимостей выполните:

```bash
npx playwright install
```

### 6. Запустите тесты

Для запуска всех тестов выполните команду:

```bash
npx playwright test
```

## Отчёт после выполнения тестов

После запуска тестов Playwright формирует HTML-отчёт.  
Открыть его можно командой:

```bash
npx playwright show-report
```
