## Страница объявлений — фильтр цены

 Общее предусловие для страницы объявлений: открыта страница `/list`

### E2E-PRICE-001

| Поле | Данные |
|---|---|
| ID | E2E-PRICE-001 |
| Заголовок | фильтрация по диапазону цен работает корректно |
| Предусловие | в выдаче есть объявления с разными ценами |
| Шаги | 1. открыть страницу объявлений<br>2. ввести значение в поле `От` = 1000 <br>3. ввести значение в поле `До` = 10000 <br>4. применить сортировку по цене по возрастанию<br>5. проверить карточки в выдаче<br>6. применить сортировку по цене по убыванию<br>7. проверить карточки в выдаче |
| Ожидаемый результат | цена каждой карточки находится в заданном диапазоне, объявления вне диапазона отсутствуют |

|---|---|

## Страница объявлений — сброс фильтров

| Поле | Данные |
|---|---|
| ID | E2E-RESET-015 |

| Ожидаемый результат | отсчёт таймера начинается заново |

### E2E-STATS-002

| Поле | Данные |
|---|---|

| Предусловие | автообновление активно |
| Шаги | 1. открыть страницу статистики<br>2. нажать кнопку отключения автообновления |
| Ожидаемый результат | кнопка меняет состояние на `Включить`, отображается текст `Автообновление выключено`, значение таймера скрыто |

### E2E-STATS-003

| Поле | Данные |
|---|---|
| ID | E2E-STATS-003 |

| Шаги | 1. открыть страницу статистики<br>2. отключить автообновление<br>3. нажать кнопку `Включить автообновление` |
| Ожидаемый результат | кнопка меняет состояние на `Отключить`, текст `Автообновление выключено` скрывается, значение таймера отображается и начинает уменьшаться |
