## BUG-PRICE-001

| Поле | Данные |
|---|---|
| ID | BUG-PRICE-001 |
| Заголовок | Отрицательное значение `-5000` сохраняется в поле `От` фильтра цены на странице объявлений при вводе минимальной цены |

| Ожидаемый результат | Поле `От` не принимает значение `-5000`: значение очищается, не сохраняется или появляется сообщение о невалидном вводе. |
| Actual result | Значение `-5000` сохраняется в поле `От`. |
| Severity | S3 Средняя (Major) |
| Priority | P1 Высокий (High) |
| Status | Новый |
| Author | Анжела А. |

## BUG-RESET-001

| Поле | Данные |
|---|---|
| ID | BUG-RESET-001 |
| Заголовок | Фильтры не сбрасываются на странице объявлений при нажатии кнопки `Сбросить` |
| Summary | На странице объявлений `/list` после нажатия кнопки `Сбросить` примененные фильтры остаются активными, и страница не возвращается в исходное состояние. |
| Предусловие | Открыта страница объявлений `/list` |
| Шаги | 1. Ввести значение `1000` в поле `От`<br>2. Ввести значение `10000` в поле `До`<br>3. Включить фильтр `Только срочные`<br>4. Выбрать сортировку по цене<br>5. Выбрать категорию `Недвижимость`<br>5. Нажать кнопку `Сбросить все фильтры` |
| Ожидаемый результат | поля цены очищены или сброшены; категория сброшена; фильтр `Только срочные` выключен; сортировка возвращена к значению по умолчанию, выдача возвращается в исходное состояние |
| Actual result | После нажатия кнопки `Сбросить все фильтры` все выбранные фильтры сохраняются. |
| Severity | S2 Высокая (High) |
| Priority | P1 Высокий (High) |
| Status | Новый |
| Author | Анжела А. |
| Reproducibility | Всегда |
| Affected test cases | E2E-RESET-015 |
